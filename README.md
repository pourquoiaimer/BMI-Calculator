## 內容說明
  本頁面主要練習JS的使用，並練習用localStorage將資料在本機空間做儲存及讀取，目前還未做響應式 

  頁面功能包括：
* 透過localStorage將BMI計算結果紀錄在本機，並可提取和刪除
* 輸入身高體重即可查詢BMI
* 不同健康等級有不同顏色呈現，並有紀錄計算日期
* 支持單筆和全部刪除
